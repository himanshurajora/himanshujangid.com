Things to Checkout

There are various things and websites that I encounter randomly and want to know more about them. Mostly they are frameworks, or tools for productivity. But they can be anything. I am going to put them on this page. So that both me and everyone else who loves me can learn about them as well. Trust me some of these things may impress you.

hapify.io

An API development platform that generates code from data models.

hapi.dev

A rich framework for building applications and services on Node.js.

wappler.io

A visual web application builder with server-side logic capabilities.

framework7.io

A full-featured framework for building iOS & Android apps with native look and feel.

yep.com

A platform for discovering local businesses and services.

cpu.land

An educational resource for understanding CPU architecture and operation.

polyglot.readthedocs.io

A library for natural language processing and machine learning in multiple languages.

jfxr.frozenfractal.com

A tool for creating sound effects for games and applications.

zotero.org

A free, easy-to-use tool to help collect, organize, cite, and share research.

amplication.com

An open-source platform for building production-ready backend applications.

altogic.com

A backend-as-a-service platform for building scalable web and mobile applications.

elysiajs.com

A TypeScript framework focused on performance, developer experience and end-to-end type safety.

typia.io

A TypeScript transformer that speeds up JSON validation by leveraging compiler metadata.

Super Resources

explorabl.es

A collection of interactive explanations and explorable examples from around the web.

cpu.land/editions/one-pager

A concise one-page guide to understanding CPU architecture and operation fundamentals.
